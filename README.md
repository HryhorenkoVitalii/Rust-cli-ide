# Terminal IDE

Terminal IDE на базе **Helix** + **Zellij** + **Yazi**.

## Зависимости

- [Helix](https://helix-editor.com/) — текстовый редактор
- [Zellij](https://zellij.dev/) — терминальный мультиплексор
- [Yazi](https://yazi-rs.github.io/) — файловый менеджер
- [ruff](https://docs.astral.sh/ruff/) — Python linter/formatter
- [pylsp](https://github.com/python-lsp/python-lsp-server) — Python LSP

## Установка

```bash
bash install-ide.sh
```

## Запуск

```bash
ide ~/myproject   # открыть IDE в папке
ide               # открыть IDE в текущей папке
```

## Структура

```
┌──────────┬────────────────────────┐
│          │        Helix           │
│   Yazi   │      (редактор)        │
│  (файлы) ├────────────────────────┤
│          │  Terminal (стек)       │
└──────────┴────────────────────────┘
```

## Хоткеи

### Редактор (Helix)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+R` | Запуск программы |
| `F5` | Дебаг |
| `Ctrl+S` | Сохранить |
| `Ctrl+Z` | Отмена |
| `Ctrl+Y` | Повтор |
| `Ctrl+X` | Вырезать строку |
| `Ctrl+C` | Копировать строку |
| `Ctrl+V` | Вставить |
| `Ctrl+A` | Выделить всё |
| `Ctrl+F` | Поиск |
| `Tab` | Следующий буфер |
| `Shift+Tab` | Предыдущий буфер |
| `Ctrl+W` | Закрыть буфер |

### Zellij

| Клавиша | Действие |
|---------|----------|
| `Alt+T` | Новый терминал в стеке |
| `Alt+H/L` | Фокус влево/вправо |
| `Alt+J/K` | Фокус вниз/вверх |
| `Ctrl+P` | Режим панелей |
| `Ctrl+T` | Режим вкладок |
| `Ctrl+N` | Режим ресайза |
| `Ctrl+Q` | Выход |

### Yazi (файловый менеджер)

| Клавиша | Действие |
|---------|----------|
| `Enter` / `l` | Открыть файл в Helix |
| `o` | Открыть в Helix (альт.) |
| `t` | cd терминала в текущую папку |
| `H` / `[` | Назад |
| `L` / `]` | Вперёд |

## Проектные конфиги (.ide/)

Создайте папку `.ide/` в корне проекта для локальных настроек запуска:

```
myproject/
  .ide/
    run.sh      ← команда для Ctrl+R
    debug.sh    ← команда для F5
  src/
  main.py
```

**`.ide/run.sh`** — что запускать по `Ctrl+R`:
```bash
python3 main.py
```

**`.ide/debug.sh`** — что запускать по `F5`:
```bash
python3 -m pdb main.py
```

Без `.ide/` — `Ctrl+R` и `F5` запускают текущий открытый файл.
